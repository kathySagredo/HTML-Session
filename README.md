# 📘 Clase de HTML - Introducción Completa

Este proyecto es un recurso de apoyo que contiene un archivo HTML bien comentado pensado para aprender los fundamentos de HTML desde cero.

## 🎯 Objetivo educativo

Este proyecto fue creado con fines educativos para estudiantes que están comenzando en el desarrollo web.

El archivo contiene:

- Comentarios explicativos
- Ejemplos reales
- Buenas prácticas básicas
- Explicaciones simples
- Introducción a semántica HTML

## ▶️ Cómo ejecutar el proyecto
- Descargar o clonar el repositorio.
- Abrir la carpeta en VS Code.
- Abrir el archivo: index.html

### Ejecutar con:
- Live Server (recomendado) o abrir directamente en el navegador

## 🛠️ Tecnologías utilizadas
- HTML5
